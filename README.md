# HR-Attrition-Analysis
Employee attrition (turnover) is a major challenge for organizations, leading to high recruitment costs and loss of institutional knowledge. This project leverages HR Analytics to identify the key drivers behind employee departures and provides data-driven recommendations to improve retention.

Key Performance Indicators(KPI's): Overall Employees, Attrition Attrition Rate, Active Employees and Average Age.
